How to run:

1. Open .imn file in CORE
2. Click the run button
3. Click on each node to open a console
4. Copy the source code to each node's path
5. On each node, type bash run.sh config <node_name>

Security Extension

To send send messages using the onion routing protocol, enter SECMSG <node_name> "message"

NOTE: The public/private key files (.pem) are stored in “/home/core” and generated at runtime.
